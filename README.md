# Uncraft Blocks

A [Minecraft](https://www.minecraft.net/) data pack to make certain blocks uncraftable to its raw material. This data pack is made for the [Celestrial Boundaries](https://discord.gg/celestrials) Pale Gardent Event and is compatible for Minecraft 1.21.6+.

## Recipes
- Wool -> 4 Strings
- Block of Quartz -> 4 Nether Quartzes
